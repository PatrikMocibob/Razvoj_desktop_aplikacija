# Razvoj_desktop_aplikacija

Cloud Estate

Patrik Močibob, Diego Banovac, Tin Car

Cloud Estate je ideja o aplikaciji koja bi lokalizirala tržište nekretnina u hrvatskoj pomoću agencija za nekretnine.

Bila bi slična njuškalu ali samo za nekretnine i imala bi puno bolji i više user friendly interface.

Također bi nudila garanciju o sigurnosti jer bi se poslovi provodili isključivo sa agencijama koje su provjerene.
